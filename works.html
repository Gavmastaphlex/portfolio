  • MIXER

    Languages & features incorporated:
    HTML, CSS, PHP, SQL, jQuery, AJAX

    I originally created this website while completing my web development diploma with "YOOBEE School of Design" here in Wellington.

    This assignment was to design, plan and create a dynamic website that queries and updates a database. Authentication techniques were to be included to display restricted content. A content management system that focused on providing an intuitive user experience for the customer was emphasized. The web application had to also use PHP5 classes and object based code structure.

    The website I created is for users to enter all the ingredients they have in their cupboards, and then the website displays all the recipes they can make from the database of stored user recipes. There is also the functionality for creating a profile in the website, which subsequently allows the creating and editing of recipes and ingredients.
